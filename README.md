# sankofa-public

Static-site host for Sankofa visualizations — weather zone maps, weekly reports, status pages — served via GitHub Pages and embedded in Notion / Slack threads.

## Layout

- `index.html` — landing page, links to live artifacts
- `weather/` — zone map + nowcast pages
- `assets/` — shared CSS / JS / images

## How it gets updated

Artifacts are generated by the Sankofa bot (private repo) and pushed here on a schedule. Do not edit generated files by hand — they will be overwritten on the next refresh.
